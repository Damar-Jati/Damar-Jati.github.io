A declarative, efficient, and flexible JavaScript library for building user interfaces. It lets you compose complex UIs from small and isolated pieces of code called “components”.

Tentang React

React (juga dikenal sebagai React.js) adalah pustaka JavaScript untuk membangun antarmuka pengguna. Pustaka ini dikelola oleh Meta dan komunitas pengembang.

Berikut adalah contoh komponen sederhana di React. Anda bisa menyimpannya dalam file berekstensi .jsx.

import React from 'react';

function Welcome(props) {
  return <h1>Hello, {props.name}</h1>;
}

export default Welcome;
